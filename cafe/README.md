giord with calsand inpu# ☕ Симуляция очереди заказов в кафе (Python)

Данный проект представляет собой консольную симуляцию работы кафе с очередью заказов.
Используется структура данных ,очередь (FIFO), и объектно-ориентированный подход.

---

## Возможности проекта:

- Очередь заказов реализована с помощью `collections.deque`
- Автоматическое добавление случайного количества клиентов
- Каждый клиент делает случайное количество заказов из меню
- Возможность ручного добавления заказа
- Клиент может выбрать несколько блюд
- Перед отправкой заказа в работу выполняется подтверждение
- Каждое блюдо готовится случайное время (от 1 до 3 секунд)
- Защита от некорректного ввода пользователя

---

## Структура проекта:

Проект состоит из одного файла и включает следующие классы:

### Order
Представляет один заказ:
- имя клиента
- название блюда

### Client
Представляет клиента кафе.

### CafeQueue
Управляет очередью заказов:
- добавление заказов
- проверка наличия заказов
- приготовление заказов по принципу FIFO

---

## Используемые технологии

- Python 3
- `collections.deque` - очередь
- `random` - генерация случайных данных
- `time.sleep()` - имитация времени приготовления
