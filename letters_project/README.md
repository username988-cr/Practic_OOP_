# Почтовый сервис (учебный проект OOP)

# Цель работы

Закрепить основы объектно-ориентированного программирования на Python:
- классы
- наследование
- методы классов
- переопределение методов

## Описание программы

Программа имитирует работу почтового сервиса.  
Пользователь может выбрать способ отправки сообщения и ввести необходимые данные.

Доступные способы отправки:
- Email
- Почта
- Telegram

Сообщение не отправляется реально, а выводится в консоль.

## Используемые классы

### Класс `Leter`
Базовый класс, в котором:
- хранятся данные отправителя, получателя и текста сообщения
- реализован метод `send()` для вывода сообщения

### Классы-наследники
- `Email`
- `Post`
- `Telegram`

Эти классы наследуются от `Leter` и используют (или переопределяют) метод `send()`.

## Принцип работы программы

1. Программа приветствует пользователя
2. Пользователь выбирает способ отправки
3. Вводит:
   - имя отправителя
   - имя получателя
   - текст сообщения
4. Сообщение выводится в консоль
5. Пользователь может отправить ещё одно сообщение или завершить программу
