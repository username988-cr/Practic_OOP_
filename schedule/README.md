# Студент, расписание и стипендия (OOP)

## Цель проекта

Учебный проект для практики объектно-ориентированного программирования на Python:
- работа с классами
- методы классов
- взаимодействие между объектами
- использование случайных данных (`random`)

## Описание программы

Программа моделирует студента с учебным расписанием, оценками и расчётом стипендии.

В программе:
- автоматически создаётся расписание на неделю
- случайным образом генерируются оценки
- считается средний балл студента
- определяется размер стипендии

Все данные выводятся в консоль.

## Используемые классы

### Класс `Student`

Отвечает за данные и действия студента.

Хранит:
- имя студента
- список возможных предметов
- расписание на неделю
- оценки по предметам

Основные методы:
- `generate_week_schedule()` — создаёт расписание на неделю (2–3 предмета в день)
- `generate_grades()` — генерирует оценки по каждому предмету
- `show_schedule()` — выводит расписание
- `show_grades()` — выводит оценки
- `average_grade()` — считает средний балл

### Класс `Stipendiya`

Отвечает за расчёт стипендии.

Метод:
- `calculate(student)` — возвращает размер стипендии на основе среднего балла

## Логика расчёта стипендии

- Средний балл **10 и выше** → 2000 грн  
- Средний балл **8–9.99** → 1500 грн  
- Средний балл **6–7.99** → 1000 грн  
- Средний балл **ниже 6** → 0 грн  

## Принцип работы программы

1. Пользователь вводит имя студента
2. Программа:
   - создаёт расписание на неделю
   - генерирует оценки
3. Выводит:
   - расписание
   - оценки
   - средний балл
   - размер стипендии
