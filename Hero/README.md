# Simple RPG Game (Python)

Небольшая консольная RPG-игра на Python, созданная в учебных целях для практики объектно-ориентированного программирования (OOP).

---

## Структура проекта

project/
- main.py — точка входа в игру
- classes.py — все игровые классы (герои, враги, бой, магия, магазин)

---

## Возможности игры

- Выбор героя (Ловкач, Силовик, Маг, Универсальный)
- Пошаговый бой с полностью рандомным врагом
- Специальные способности для каждого класса героя
- Магия (ограниченное количество заклинаний)
- Магазин с предметами
- Рюкзак (инвентарь)
- Критические удары
- Уклонение (у ловкача)
- Рандомное количество монет у героя
- Награда в монетах за победу над врагом

---

## Герои

- **Ловкач** — может уклоняться от нескольких атак врага  
- **Силовик** — повышенный урон и мощный удар  
- **Маг** — увеличенный запас маны и магические атаки  
- **Универсальный** — сбалансированный герой с лечением  

---

## Бой

Во время боя доступны действия:
1. Атака  
2. Спец-умение  
3. Магия  
4. Использовать предмет  
5. Показать рюкзак  

При неверном вводе выполняется обычная атака.

---

## Магазин и предметы

В магазине можно купить:
- Эликсир здоровья
- Эликсир маны
- Улучшение урона

Предметы применяются из рюкзака по номеру.

---

## Технические особенности

- Используется ООП (классы, наследование, абстрактные классы)
- Игра не падает при неправильном вводе
- Проект разбит всего на 2 файла
